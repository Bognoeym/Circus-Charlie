# Circus Charlie
WINAPI로 제작한 고전게임 서커스 찰리 모작이다. 매프레임마다 계산되어 그려지는 화면 전환을 보완하기 위한 더블 버퍼링 기술을 익히기 위해 제작하였다.
- WINAPI(C++), Visual Studio
- 서커스 찰리 게임의 첫 번째 스테이지 모작
- 플레이 영상 :: 

## 조작법
- **←, →**: 좌, 우 이동
- **Spacebar**: 점프
- **Enter**: 선택
